# dynamic-qrcode-image

安装

```sh
pip install dynamic-qrcode-image
```

- 模块导入

```py
from dynamic_qrcode_image import QRC
```

- 本地运行

```sh
python -m dynamic_qrcode_image
```

```sh
qrcode-server
```
